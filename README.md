# Alg Sets

Web app for storing and exploring cubing algorithms. Check out [the website](https://algsets.jonatanklosko.com/).

## Development

Node and MongoDB are required.

```bash
git clone https://github.com/jonatanklosko/alg-sets.git && cd alg-sets
```

**Server**

```bash
cd server
npm install
npm start
```

**Client**

```bash
cd client
npm install
npm start
```
